is

A dispatcher + a filesystem

  • A tiny SKILL.md (~2 KB) that tells Claude when to invoke + the script surface
  • Shell scripts in bin/ that do the byte-pushing
  • A snippet library in snippets/ — one file per element
  • A theme library in themes/ — one CSS file per theme
  • A per-page JSON manifest as source of truth; HTML is rendered from it
isn't

A diagramming tool or build system

  • No runtime renderers (Mermaid, D3, Chart.js) — inline SVG is first-class
  • No ASCII-art diagrams — wireframe with HTML/CSS
  • No build step, no bundler, no asset pipeline
  • Snippets ship as structural shells; data values are slot-fills
  • Not a usage-tracking system — promotion is manual
